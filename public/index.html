Here are some "tools" that I made which can be useful if you want a single thing done.

Flash Cards
A tool that can be used to help memorize things. Record "question-answer" pairs
List randomizers
Random From List
Random Remover
List Randomizer
Several similar little tools that help make up your mind for you when you need to make a decision.
Soundz
A toy for generating sounds with an Oscillator in your browser. Open several windows and jam out.
SVG Test
This can be used to test making SVG images, it lets you view the SVG image you're working on on the right of the page.

Here are some userscripts that I made for fun or utility. To install them in Firefox, just click the links, in Chrome you need to download the file and drag it into your extensions page.

No Typing Notifications
This lets you go more incognito on Facebook. It disables some chat features such as the "typing" notification and the "seen" notification.
Autopoke
This is a simple autopoke script. If you open your pokes page it will automatically start responding to pokes for you. To stop it just navigate to another page.
XKCD News Substitutions
This script changes certain key words in text to make it more fun to read. It works by modifying text nodes and won't break a site's HTML. Idea: XKCD
